# DRMacIver's Junk Drawer

This is a repository for code that I write but don't have any plans to release formally or actively maintain.

This is a similar strategy to the one I outlined in [Why am I not working on my PhD?](https://drmaciver.substack.com/p/why-am-i-not-working-on-my-phd): This repository basically exists as a platform to allow me to have fun tinkering with code, lowering the activation energy of trying new things and having fun hacking on stuff.

It's mostly stuff that I've written because it would be interesting to write, or because I thought it would enable to me to write something else interesting. I may also add stuff here that is actually useful at some point, but only if it's fun.

Code in here should generally be of adequate quality and you are welcome to use it through the magic of copy and paste, but I make no promise that it works, I've probably never run it in production, and I likely won't support it if you find problems and may not even accept pull requests. Feel free to read it for its interest value, but use it at your own peril and I don't promise not to laugh if this bites you.
